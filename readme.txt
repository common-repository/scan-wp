Scan WP is a free tool you can use on your own site on any page you wish. Engage your users while increasing time on site and reducing bounce rate.

== Description ==

Allow visitors to scan competitors websites and know what theme and plugins they are using.
The results will contain the theme name, price, version, download link, plugin names, plugin price and download links.

If your users are looking to build a website or rebrand their existing one, Scan wp plugin is perfect for you! 
This plugin is the Wordpress plugin version of the known [Scan WP](http://scanwp.net "Scan WP") Theme and plugin detector.
The plugin is FREE of charge and can be implemented on any page or post on your site.
If you are looking for something that will keep your users engaged and interested with your site, look no further. 
Implementing this free tool will increase your users' enganement while improving your site's overall experience. 
A huge SEO plus for you!

== Installation ==

The best practice to install the plugin is:

1. Download the plugin
2. In your admin panel, go to Plugins -> Add New
3. Click "Upload Plugin" and choose the Zip file you downloaded
4. After the plugin is uploaded click "Activate plugin"

If you rather do things manually:

1. Upload 'scan-wp' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Implementing ==

In order to implement the tool on your page or post, just paste<code>[scan-wp]</code> into your post or page.
After installation, an explanation page will be added under the plugins tab in the admin area.
